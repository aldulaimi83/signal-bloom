# Signal Bloom

Signal Bloom is a speculative monitoring interface that treats world activity like a living instrument instead of a spreadsheet. It is a single-page static experience with a cinematic hero, procedural signal field, generated anomaly feed, and mood controls that let the viewer reshape the atmosphere in real time.

## What it is

- A bold static web experience designed to feel strange, polished, and alive
- A lightweight front-end artifact that can be hosted almost anywhere
- A playful concept piece exploring emotional interface design for global monitoring

## Run it

Open `index.html` in a browser.

## Files

- `index.html` contains the structure
- `style.css` defines the visual system and motion
- `script.js` generates the animated signal field and synthetic feed
