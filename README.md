# Signal Bloom

Signal Bloom is a speculative monitoring interface that treats world activity like a living instrument instead of a spreadsheet. It is a static single-page experience with a cinematic hero, procedural signal field, generated anomaly feed, scenario presets, a memory archive, and an operator transmission composer.

## What it is

- A bold static web experience designed to feel strange, polished, and alive
- A GitHub Pages-friendly front-end artifact with manifest, favicon, and fallback page
- A playful concept piece exploring emotional interface design for global monitoring

## Run it

Open `index.html` in a browser.

## Deploy

This project is static, so it can be deployed directly with GitHub Pages from the repository root.

## Files

- `index.html` contains the structure and metadata
- `style.css` defines the visual system, layout, and motion
- `script.js` generates the animated signal field and interactive layers
- `site.webmanifest`, `favicon.svg`, and `404.html` support a more deployment-ready static site
